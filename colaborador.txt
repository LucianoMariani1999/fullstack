
###########
Pasos para compartir un Proyecto en Github
###########

1. Un miembro del equipo crea un proyecto y un repo.

2. Agrega colaboradores en:
   
      -Settings
      -Colaboradores
      -Add people con el mail de compañero/usuario de Github

3. Realiza un git clone en la terminal .....url del proyecto

    Git Clone https://github.com/Bernipas/repocompartir.git 

4. Desarrollamos código

5. Realizamos un git pull ==> Verifica si hay cambios en el repo remoto y los descarga.

6. Realiza un git push ... url del repo
